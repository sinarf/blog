The main goal of this project is to be part of eclipse, for now it's on a the proposal stage.

There is already 2 projects to integrate maven into eclipse IDE, m2eclipse is the more advance. The other one is integrated in maven in just consist by calling eclipse goal, it works but I'm not sure you can call that integration. ;)

I'd love to see a good integration of maven in eclipse, as netbeans already has.
